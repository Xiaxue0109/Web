Determines a grid item’s location within the grid by referring to specific grid lines. grid-column-start/grid-row-start is the line where the item begins, and grid-column-end/grid-row-end is the line where the item ends. Values: – can be a number to refer to a numbered grid line, or a name to refer to a named grid linespan – the item will span across the provided number of grid tracksspan – the item will span across until it hits the next line with the provided nameauto – indicates auto-placement, an automatic span, or a default span of one
